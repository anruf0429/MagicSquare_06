| 테스트 일자 | (실행 시 기록) |
| 우선순위 | P1 |

**테스트 목적**

0이 정확히 두 개인 일반 격자에서 빈칸 좌표가 row-major 순서로 수집되는지 검증한다.

**테스트 기능**

빈칸 찾기 (find_blank_coords)

**입력값**

4×4 정수 행렬. `0` 위치가 (1,3), (3,3)인 예시 격자(좌표 표기는 명세/구현의 0-index·1-index 정책에 맞출 것).

**테스트 단계 (Given / When / Then)**

- **Given** PRD에 부합하는 4×4 행렬이 주어진다.
- **When** `find_blank_coords`(또는 동등 기능)를 호출한다.
- **Then** 반환 리스트는 `(1,3)`, `(3,3)` 순서(또는 설계상 정한 인덱싱에 따른 동일 위치)여야 한다.

**테스트 환경**

Python 3.x, pytest (프로젝트에 `pyproject.toml` 도입 시 동일 환경으로 고정)

**전제 조건**

행렬 크기 4×4, 값은 0 또는 1~16, 비0 중복 없음 등 경계 검증을 통과한 상태.

**성공 / 실패 기준**

반환 좌표가 예상과 일치하면 성공.

**특별하게 필요한 절차**
